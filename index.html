CSS Typography Cheat Sheet

CSS Property Values Description

color

 Defines the color of the text

font-family

 When using multiple values, the font-family list of font families defines the priority in which the browser should choose the font family.

font-weight

 In CSS, the font-weight property controls how bold or thin text appears. It can be specified with keywords or numerical values.

bold

 Bold font weight.

normal

 Normal font weight. This is the default value.

lighter

 One font weight lighter than the element’s parent value.

bolder

 One font weight bolder than the element’s parent value
